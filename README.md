# PalindroneCheckerApp

# Objective
     #Build a simple tool that detects if a word or sentence reads the same backward as forward.
# Use Case
     #  The app ignores spaces and grammar to check if the core letters form a perfect mirror image.
# Project Flow
    # Input Clean-up: Strip out the junk like commas and spaces so only letters remain.
    #  Comparison: Look at the first and last letters and work toward the middle.
    #   Result: Return a quick "true" if it matches or "false" if it doesn't.
